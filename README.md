# 🎓 Course Registration Management System (CRMS)

## 📌 Project Description

The **Course Registration Management System (CRMS)** is a web-based application developed using **Java Spring Boot** that allows students to register for courses and enables administrators to manage courses, programs, and student data efficiently.

This system simplifies the manual process of course registration by providing an automated and user-friendly platform.

---

## ✨ Features

### 👨‍🎓 Student Features

* Student Registration & Login
* View Available Courses
* Register for Courses
* View Registered Courses
* View Timetable

### 👨‍💼 Admin Features

* Admin Login
* Add / Edit / Delete Courses
* Add / Manage Programs
* View Registered Students
* Manage Course Enrollment

---

## 🛠️ Tech Stack

* **Backend:** Java, Spring Boot
* **Frontend:** HTML, CSS, Thymeleaf
* **Database:** MySQL

## ⚙️ How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/Ritesh02189/Course-Registration-Management-System-.git
   ```

2. Open the project in IntelliJ / Eclipse

3. Configure MySQL database:

   * Create database
   * Update `application.properties` with DB username & password

4. Run the project:

   ```bash
   ./gradlew bootRun
   ```

5. Open browser:

   ```
   http://localhost:8080
   ```
